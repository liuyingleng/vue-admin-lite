# Vue Admin Lite

A lightweight admin dashboard template based on shadcn-vue, Vue3 and Vite.

## Features

- ⚡️ Vue 3 + Vite 7 - Fast and modern
- 🎨 shadcn-vue + Tailwind CSS - Fully customizable UI
- 🔐 Route-based authentication
- 📱 Responsive design
- 🌙 Dark mode support
- 📦 Minimal dependencies

## Tech Stack

- Vue 3.5 + TypeScript
- Vite 7
- Pinia (State Management)
- Vue Router 5
- shadcn-vue (UI Components)
- Tailwind CSS 4

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Project Structure

```
src/
├── layouts/          # Layout components
├── pages/            # Page components (auto-routing)
├── components/       # UI components
├── stores/           # Pinia stores
├── router/           # Router configuration
└── utils/            # Utility functions
```
